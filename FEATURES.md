# Implemented product features

## Agent governance
- Scoped, expiring and revocable permits
- Per-action and cumulative budgets
- Target allowlists and rate limits
- Agent-level and workspace-level emergency stops

## Human control
- Approval inbox for escalated actions
- Approve/deny decisions with reviewer identity and notes
- Incident timeline and resumable workspace state

## Security
- Deterministic prompt-injection firewall before policy evaluation
- AES-256-GCM credential vault with metadata-only API responses
- Public-HTTPS-only protected execution with SSRF protections
- Signed, hash-chained receipts and verification API

## Product experience
- Visual condition-based policy builder
- Live security event feed and risk center
- Agent directory, permits, policies, receipts and approvals
- Rialo adapter status and anchored governance events
